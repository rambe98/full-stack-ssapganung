css
- css는 html의 각 요소의 스타일을 정의하여 화면등에 
어떻게 렌더링하면 되는지 브라우저에게 설명하기 위한 언어
- cascading sytle sheet의 약자임
- 색상 폰트 각요소의 위치 변경 백그라운드 및 간단한 애니 횩과를 줄 수 있음

css선택자 
1. 태그의 이름으로 선택

2. 클래스 선택자
- class 속성: 특정 집단의 여러 요소를 한번에 선택할 때 사용

3. 아이티 선택자
-  id 속성 : css 를 적용할 특정 요소를 선택할 때 사용
- 문서당 id 값은 한번만 사용

html에 css를 적용하는 법
1. 인라인 방식
 - 태그 내에서 style 속성에 직접 작성하는 방식
2. 내장 방식
- 같은 html 문서 내 style 태그 안에 스타일을 적용하는 방식
3. 링크 방식
- html 외부에 css 파일을 만들고 불러와 적용하는 방식

선택자 - 어디에적용을 하는지
css 문법- 뭘 적용 시킬건지
적용 방식 - 어떻게 적용 시킬 건지

스타일의 중첩
- 일반적으로 동일한 선택자를 대상으로 여러개의 css명령어가 입력되면
모든 명령어가 적용된다.
- 적용 하려는 스타일이 서료 다를 때만 성립한다.

스타일이 중첩되어 생길 수 있는 문제점
- css 코드가 중복될 경우 항상 마지막에 입력된 명령이 실행된다.

색상을 표현하는 방법
css에서는 배경색 글자색 테두리의 색등과 같이 색상을 다루는 경우가 자주 있다

inherit
영단어로 상속이라는 의미다
스타일 값으로 inherit를 지정하면 부모 태그의 색상을 그대로 물려받는다.

컬러네임
red, green등 색상을 의미하는 단어

rgb 또는 rgba코드
rgb 코드는 색상을 표현하는 가장 단순한 코드임
(255,255,255) 와 같이 3개의 숫자가 묶여있는 순서쌍으로 정의된다
각각 빛의 삼원색인 빨강 초록 파란색을 의미 함

rgba는 투명도를 포함한다
색상을 의미하는 rgb뒤에 투명도를 의미하는 알파값이 추가된 형태

16진수 컬러 코드(hex rgb)
색상을 16진수로 표현하는 것으로 컴터와 디스플레이의 발달로 사용도가 높아지고 있다

글자를 표현하는 스타일
html의 본질은 정보전달이며 정보 전달의 핵심은 텍스트이다
따라서 css에서는 텍스트를 아름답게 표현하기 위한 온갓 도구들이 준비되어 있음

공간을 표현하는 스타일
공간의 단위
css에서 주로 사용되는 공간의 단위에 대해 알아보자
1. % >> 부모대비 비율
2. auto >> 브라우저가 자동으로 계산하게 만드는 값
3. vw,vh >> 브라우저 창의 해상도를 기준으로 삼는 길이의 단우
 ㄴ 100vw 는 화면 너비의 100% 30vh는 화면 높이의 30%
 ㄴ 화면 크기에 상관없이 일정한 비율로 화면에 표시할 수 있다

크기 지정하기
일반적으로 크기를 지정할 때는 width와 height속성을 활용한다

여백 지정하기
css에 여백은 외부여백과 내부여백 두가지가 있다

1.외부여백(margin)
요소 바깥쪽의 여백, 여백에는 다른 요소가 침범할 수 없음

2. 내부 여백(padding)
요소 안쪽의 여백을 의미


영역을 표시하는 스타일
영역의 테두리를 표시하는 border

선의 종류 - border
- border 스타일 속성에는 선의 종류를 입력합니다.
- 선을 없애고 싶다면 스타일 속성 자체를 삭제하거나
스타일 값으로 none을 입력한다
- 점선, 대쉬, 실선, 겹선등 다양한 종류의 선을 적용할 수 있다.

화면에 표시되는 방식을 표현하는 스타일 visibility, display
표시하거나 숨기는 기능 - visibility
회면에 표시될 규격을 정하는 display
block: 한 행을 다 차지한다.
inline: 요소 크기만큼의 공간을 차지한다.

요소의 배치
display : grid 속성
grid 스타일은 css를 사용해 일종의 격자를 그리는 기술이다.
그리드를 적용하려면 div태그를 대상으로 display:grid 스타일을 적용하면 됨
