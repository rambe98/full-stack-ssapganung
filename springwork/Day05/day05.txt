findBy : 특정 조건에 맞는 데이터 조회
countBy : 특정 조건에 맞는 데이터 개수 조회
deleteBy : 특정 조건에 맞는 데이터 삭제
existsBy : 특정 조건에 맞는 데이터 존재 여부 확인

메서드 명에 다양한 조건자를 함께 사용할 수있음
And :  두 조건을 모두 만족하는 데이터 조회
Or: 두 조건 중 하나라도 만족하는 데이터 조회
Between : 두 값 사이에 있는 데이터 조회
less Tham / Greate Than : 특정 값보다 작은 또는 큰 데이터 조회
Like : 부분 일치 검색
